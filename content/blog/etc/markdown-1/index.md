- **굵은 글씨**<br/>
    ```
    **굴게**
    __굵게__
    ```

    ```
    ~~정정하기~~
    ```

## 각주 (Footnotes)
> 각주[^fn] 

각주는 너무 길어 본문에 기입하기 적당하지 않거나 본문 내용에 끼워 넣었을 때 문맥의 흐름에 방해될 때 사용된다<br/>
각주를 사용하고자 하는 문자열 뒤에 ``[^fn1]``와 같이 각주 참조를 넣고, ``[^fn1]:첫번째 각주 내용``와 같이 각주 내용을 적어준다.

```
첫번째 각주[^fn1] 두번째 각주[^fn2]

[^fn1]:
  첫번째 각주 내용
[^fn2]:
  두번째 각주 내용
```
