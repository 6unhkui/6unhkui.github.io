두 개 뽑아서 더하기

정수 배열 numbers가 주어집니다. numbers에서 서로 다른 인덱스에 있는 두 개의 수를 뽑아 더해서 만들 수 있는 모든 수를 배열에 오름차순으로 담아 return 하도록 solution 함수를 완성해주세요.

문제 자세히 보기 →

나의 풀이

function solution(numbers) {
    const res = numbers.reduce((prev, cur, i, arr) => {
        for (let j = i + 1; j < arr.length; j++) {
            prev.add(cur + arr[j]);
        }

        return prev;
    }, new Set());

    return [...res].sort((a, b) => a - b);
}

인수로 전달되는 numbers 배열을 Array.prototype의 reduce 메서드로 순회하면서 서로 다른 index에 있는 값과 더하고, 그 값의 중복을 피해기 위해 Set 객체에 넣어주었다. Set은 요소 값이 중복 되지 않고 순서를 보장하지 않는 다는 특징이 있다. 순회 가능한 데이터 컬렉션으로, 이터레이션 프로토콜을 준수하는 이터러블이므로 스프레드 문법을 사용 할 수 있다. 스프레드 문법으로 Set 객체를 배열로 변환한 후, sort 메서드를 사용해 오름차순을 정렬하도록 코드를 작성하였다.

  • Array.prototype의 sort 메서드는 배열의 요소를 일시적으로 문자열로 반환한 후 유니코드 코드 포인트의 순서를 기준으로 정렬한다. 그렇기 때문에 콜백 함수로 정렬 순서를 정의하는 비교 함수를 전달해야 숫자의 제대로 된 정렬이 이뤄진다.

완주하지 못한 선수

수많은 마라톤 선수들이 마라톤에 참여하였습니다. 단 한 명의 선수를 제외하고는 모든 선수가 마라톤을 완주하였습니다. 마라톤에 참여한 선수들의 이름이 담긴 배열 participant와 완주한 선수들의 이름이 담긴 배열 completion이 주어질 때, 완주하지 못한 선수의 이름을 return 하도록 solution 함수를 작성해주세요.

문제 자세히 보기 →

나의 풀이

function solution(participant, completion) {
    participant.sort();
    completion.sort();

    return participant.find((v, i) => v !== completion[i]);
}

participant 배열에는 존재하지만 completion 배열에는 존재하지 않은 요소가 완주하지 못한 선수이므로 두 배열을 동일한 방법으로 정렬 한 뒤, 같은 index에 위치한 값이 다르면 그 값을 리턴하도록 코드를 작성하였다.

K번째수

배열 array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수를 구하려 합니다. 배열 array, [i, j, k]를 원소로 가진 2차원 배열 commands가 매개변수로 주어질 때, commands의 모든 원소에 대해 앞서 설명한 연산을 적용했을 때 나온 결과를 배열에 담아 return 하도록 solution 함수를 작성해주세요.

문제 자세히 보기 →

나의 풀이

function solution(array, commands) {
    return commands.map(command => {
        const [i, j, k] = command;
        return array.slice(i - 1, j).sort((a, b) => a - b)[k - 1];
    });
}

2차원 배열 commands를 map 함수로 순회하면서 그 요소인 1차원 배열의 값을 순서대로 배열 디스트럭처링 할당 문법을 통해 i, j, k 변수에 담았다. 그리고 Array.prototype의 slice 메서드를 사용해 array 값을 i번째 요소부터 j번째 요소까지 복사한 뒤 그 값을 정렬 하고, 정렬한 값에서 k 번째 요소를 리턴하도록 코드를 작성하였다. 배열의 index는 0번부터 시작하므로 순서로 값에 접근할 경우 -1을 해줘야 하지만, slice 메서드에서 두번째 인자로 전달하는 복사를 종료할 index 값은 그 index에 해당하는 요소를 복사 대상에 포함하지 않으므로 -1을 하지 않아도 된다.

2016년

2016년 1월 1일은 금요일입니다. 2016년 a월 b일은 무슨 요일일까요? 두 수 a ,b를 입력받아 2016년 a월 b일이 무슨 요일인지 리턴하는 함수, solution을 완성하세요. 요일의 이름은 일요일부터 토요일까지 각각 SUN,MON,TUE,WED,THU,FRI,SAT 입니다. 예를 들어 a=5, b=24라면 5월 24일은 화요일이므로 문자열 TUE를 반환하세요.

문제 자세히 보기 →

나의 풀이

function solution(a, b) {
    const day = new Date(2016, a - 1, b);
    const dayNames = ["SUN", "MON", "TUE", "WED", "THU", "FRI", "SAT"];
    return dayNames[day.getDay()];
}

Date 생성자 함수의 인자로 연, 월, 일, 시 등 날짜를 나타내는 요소 값을 전달해 Date 객체를 생성 할 경우, 두번째 인자인 month 값은 0부터 시작하므로 -1을 해줘야 한다. (0 = 1월) Date.prototype의 getDay 메서드는 요일(0 ~ 6으로, 0 = 일요일)을 나타내는 정수를 반환한다. 그 순서에 맞게 각 요일의 문자열을 담은 배열을 생성하고, Date 객체의 getDay 메서드의 반환 값 위치에 있는 요일 문자열을 리턴하도록 코드를 작성하였다.
